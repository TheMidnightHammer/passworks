# passworks
Not a secure way to store pass words with encryption.
This was only a very small project, i made it because i was bored. I will be updating this in the future.
I will maybe add a GUI interface later, using Hammer when it will have font rendering. 
## Building
Run this command in the passworks folder:
`g++ passworks.cpp -o passworks`

## Running
After building just run `./passworks`.
